# 保育メモ (hoiku-app)

保育士が子どもの日々の様子を簡単に記録できるWebアプリです。  
園児ごとの体調や生活記録を管理し、日々の保育記録を効率よく残すことを目的としています。

---

## 主な機能

- **ログイン機能**  
  8桁のパスワードで職員認証を行い、アプリへログインできます。

- **子ども登録機能**  
  園児の情報を登録し、新しい子どもを追加できます。

- **子ども一覧表示**  
  登録された園児をカード形式で一覧表示します。

- **記録入力機能**  
  体調・ごはん・おやつ・午睡・排便などのチェック項目とメモを入力し、園児の記録を登録できます。

- **出欠管理機能**  
  出席・欠席・遅刻・早退の区分を選択し、必要に応じて理由を入力できます。

- **記録一覧表示**  
  園児ごとの記録を新しい順に一覧表示し、編集・削除が可能です。

- **自動下書き保存**  
  入力途中のデータをローカルストレージに自動保存し、ページ離脱時の入力データ消失を防ぎます。

- **React 19**
- **Tailwind CSS v4**
