# 保育メモ（hoiku-app）

保育士が簡単に子どもの様子を記録できるメモアプリです。

---

## アプリ概要

保育士が子どもの出欠や体調、食事、午睡などを簡単に記録し、  
一覧で確認・管理できるアプリです。

---

## 対象ユーザー

保育士が日々の記録を短時間で効率よく管理するためのアプリです。

---

## 機能

- **ログイン**：8桁パスワードで職員認証
- **子ども一覧**：登録された子どもをカード表示
- **記録入力**：体調・食事・午睡・排便などを選択＋メモで記録
- **出欠管理**：出席 / 欠席 / 遅刻 / 早退 の区分入力
- **記録一覧**：子どもごとの記録を新しい順に表示、編集・削除対応
- **フィルター機能**：出欠区分で記録を絞り込み表示
- **削除確認**：削除前に確認ダイアログを表示
- **メッセージ表示**：成功・失敗時に画面上に表示
- **自動下書き保存**：入力中のデータをローカル保存
- **AI要約機能**：記録内容をもとに1日のまとめを自動生成

---

## 技術スタック

- Next.js（App Router, TypeScript）
- React
- Tailwind CSS
- Supabase（PostgreSQL）

---

## データ管理

- Supabaseを使用してデータを管理
- 子どもごとの記録をDBに保存し、一覧・編集・削除が可能

---

## セットアップ

```bash
npm install
cp .env.local.example .env.local
# .env.local に Supabase の URL と anon key を設定
npm run dev
```
